Como rodar código em C++?


Todo código feito para C++ usa a versão C++14. Para compilar execute o seguinte comando:

  g++ -std=c++14 <INPUT_FILE_NAME> -o <OUTPUT_FILE_NAME>

E para rodar o código e verificar os resultados:

  ./<OUTPUT_FILE_NAME>

Como rodar código em Kotlin?


Para compilar código em Kotlin use o seguinte comando:

  kotlinc <INPUT_FILE_NAME> -include-runtime -d <OUTPUT_FILE_NAME>.jar

E para executar o código:

  java -jar <OUTPUT_FILE_NAME>
